# Requisitos do sistema

## Requisitos funcionais

### RF01 — Cadastro de animais
O sistema deve permitir cadastrar animais com brinco, nome/apelido, raça, sexo, data de nascimento e lote.

### RF02 — Registro de pesagens
O sistema deve permitir registrar pesagens por animal.

### RF03 — Registro sanitário
O sistema deve permitir registrar eventos sanitários por animal.

### RF04 — Alertas sanitários
O sistema deve exibir alertas de vacinas pendentes.

### RF05 — Dashboard zootécnico
O sistema deve calcular total de cabeças, status do rebanho e evolução de peso.

### RF06 — Controle financeiro
O sistema deve registrar receitas e despesas.

### RF07 — Balanço financeiro
O sistema deve gerar balanço financeiro básico.
